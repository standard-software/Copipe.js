# Copipe.js
simple code parts

# How to use

## type CommonJS

```
// index.js
const Copipe = require("@standard-software/copipe");
Copipe();
```
	node index.js

## type ES Modules

```
// index.mjs
import Copipe from '@standard-software/copipe';
Copipe();
```
	>node --experimental-modules index.mjs
	(node:xxxxx) ExperimentalWarning: The ESM module loader is experimental.
